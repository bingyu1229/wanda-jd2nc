# Kingdee 科目表：HTM → CSV 与列 A 规范化

## 1. 将 HTM 转为 CSV（已完成逻辑）

- 使用脚本：`test/htm_table_to_csv.ts`
- 在项目根目录执行（需已 `npm install`）：

```bash
npx tsx test/htm_table_to_csv.ts "test/015.001/昆明万达房地产有限公司2002科目.htm"
```

- 默认输出：与输入同路径、同主文件名、扩展名为 `.csv`（例如 `…2002科目.csv`）。
- 转换时会 **跳过 HTML 表内第 1 行**（`第1列`…`第18列` 占位表头），**删除第 3 列（助记码）**；CSV 首行为 `科目代码`、`科目名称`、… 等业务表头。
- 可选参数：
  - `-o <路径>`：指定输出 CSV
  - `--encoding gb18030`：输入编码（默认 gb18030，兼容 gb2312）
  - `--utf8-bom`：输出带 UTF-8 BOM，便于 Windows 下 Excel 打开
  - `--excel-text-cols 0,1`：指定列以 Excel 文本公式形式写出
  - `--no-normalize-col-a`：跳过 **第 2 节** 列 A 的全部处理（2.1–2.4），列 A 保持 HTM 原样

说明：转换阶段不按数字解析单元格，避免日期/期间等文本被截断或丢末尾 `0`。

## 2. 对列 A（科目代码）规范化（同一操作对象）

仅对「整格为数字与点号组成」的科目代码做改写；**表头、中文、非点分数字**等保持原样。  
为排除金蝶 **期间** 与点分日期，形如 `YYYY.M`、`YYYY.M.D`（19xx/20xx）的单元格 **不** 做改写。

### 2.1 规则摘要：`4.(3)*`（带点）

- **第一段**（第一个 `.` 之前）：固定为 **4 位数字**，不足时在 **右侧** 用 `0` 补齐；超过 4 位时取前 4 位，**剩余数字拼接到下一段**（与下一段原内容首尾相接后再切分）。
- **每一段**（第一个 `.` 之后的每一个「点与点之间」的片段）：在该片段内部从左到右每 **3** 位一组；该片段最后一次不足 3 位时在 **右侧** 用 `0` 补满 3 位。  
**不会** 把多个片段的数字先拼成一条再按 3 位切（否则 `1151.13.01` 会错误变成 `1151.130.100`）。
- **仅一段且无后续层级时**：若只有第一段 4 位、后面没有任何 3 位组，则 **只保留这 4 位**，不再追加 `.000` 等。

### 2.2 带点形态示例（2.1 的结果，仍含 `.`）


| 原始 A 列            | 带点结果                   |
| ----------------- | ---------------------- |
| `1001`            | `1001`                 |
| `123`             | `1230`                 |
| `1001.001`        | `1001.001`             |
| `1002.01`         | `1002.010`             |
| `1151.13.01`      | `1151.130.010`         |
| `123.34.456.3456` | `1230.340.456.345.600` |


### 2.4 去掉小数点（最终写入 CSV 的列 A）

在得到 **2.1–2.2 的带点串** 之后，**删除所有 `.`**，得到连续数字串（便于部分导入或匹配场景）。

示例：`1234.567.890` → `1234567890`。


| 原始 A 列            | 最终 A 列（无 `.`）      |
| ----------------- | ------------------ |
| `1001`            | `1001`             |
| `123`             | `1230`             |
| `1001.001`        | `1001001`          |
| `1002.01`         | `1002010`          |
| `1151.13.01`      | `1151130010`       |
| `123.34.456.3456` | `1230340456345600` |


（先按 2.1 得到带点串，再按 2.4 去掉全部 `.`。）
